# Task Management Application

A simple full-stack Task Management application built using **React**, **Go (Gin)**, and **SQLite**.

This project allows users to manage tasks by performing basic CRUD operations. Users can create new tasks, view all tasks, edit existing tasks, and delete tasks. The frontend communicates with the backend through REST APIs, and all task data is stored in a SQLite database.

## Tech Stack

* React
* Go (Gin)
* SQLite
* CSS

## Features

* Create Tasks
* View Tasks
* Edit Tasks
* Delete Tasks
* Modal-based task list
* SQLite database integration
